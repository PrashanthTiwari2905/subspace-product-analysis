## 👤 About This Submission

This is a product teardown of **Subspace.money** — India's first subscription management and group sharing platform.

The teardown is based on **direct hands-on usage** of the Android app, including sign-up, exploring Public Groups, My Subscriptions, the Home dashboard, the Gift Card Marketplace, and the Group Joining flow.

## 🔍 5 Product Feedbacks at a Glance

| # | Area | Observation |
|---|------|-------------|
| 1 | UX | No onboarding — new users land directly on Public Groups with zero context |
| 2 | Features | No auto-detection of existing subscriptions — core value prop is hidden |
| 3 | UX | Multiple screens render as blank black cards with no loading feedback |
| 4 | GTM / ICP | Home dashboard is cluttered, serves too many user types at once |
| 5 | Competitor / Collaboration | Gift card discounts (0.35%–2%) too weak to compete with CRED / credit card rewards |

---

## 🧱 Framework Used

Each feedback follows the structure:

- (a) Observed — Specific screen, flow, or copy seen in the app
- (b) Problem — The real user or business pain it causes
- (c) Ship Instead — A specific, actionable solution
